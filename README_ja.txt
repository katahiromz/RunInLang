# 「言語を指定して実行」 by 片山博文MZ

## これは何？

このフリーソフトは、特定の言語でコマンドラインを実行できます。

## 使い方

1. "RunInLang" をダブルクリックします。
2. 実行したいパスファイル名またはコマンドラインを指定するか、実行したいプログラム ファイルをドロップします。
3. 言語を指定します。
4. [その言語で実行]ボタンをクリックします。指定された言語でコマンドラインが実行されます。

## 注意

- OSに対象の言語をインストールする必要があります。
- アプリがその言語をサポートしている必要があります。
- アプリの実装によっては言語が変更できないことがあります。

## 対応環境

Windows XP/Vista/7/10/11
