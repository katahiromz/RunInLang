# 「言語を指定して実行」 by 片山博文MZ

このフリーソフトは、特定の言語でコマンドラインを実行できます。

# 使い方

1. ファイル "RunInLang.exe" をダブルクリックします。
2. プログラムの名前またはコマンドラインを指定します。あるいはこのプログラムにプログラムファイルをドロップします。
3. 言語を指定します。
4. [OK]ボタンをクリックします。指定された言語でコマンドラインが実行されます。

※ 使用する言語は事前にWindowsにインストール済みにする必要があります。
※ アプリがその言語をサポートしている必要があります。
※ アプリの実装によっては言語が変更できないことがあります。

# 対応環境

Windows XP/Vista/7/10/11
